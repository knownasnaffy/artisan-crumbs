# Artisan Crumbs

A demo website for a fictional artisan bakery. Built with Next.js, React, TypeScript, and Tailwind CSS, it showcases a modern, responsive storefront complete with a hero section, product listings, and a warm, handcrafted aesthetic.

## Development

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or later
- A package manager — [Bun](https://bun.sh/) was used throughout development and is recommended, though npm, pnpm, or yarn will also work

### Setup

Install dependencies:

```bash
bun install
# or: npm install
```

### Running locally

Start the development server:

```bash
bun run dev
# or: npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

### Other commands

| Command | Description |
|---|---|
| `bun run build` | Create a production build |
| `bun run start` | Start the production server |
| `bun run lint` | Run ESLint |
